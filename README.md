## Project Plan

1.	VSLAM with keyboard/joystick control (ORB-SLAM 2)
2.	Navigation with map (simulation)
3.	Navigation with map (drone)
4.	VSLAM and Navigation without map (simulation)
5.	VSLAM and Navigation without map (drone)
